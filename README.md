# Sparklet - 闪存 - 转桌面软件了🎇 - 0.2.0

一个轻量的桌面笔记工具，支持深浅主题、多笔记管理、回收站和颜色标记，目前仅支持中文版（以后会加）

## 功能
- ✨ 快速记录灵感
- 🌓 深浅主题切换
- 📒 多笔记管理（新建、保存、删除、回收站）
- 🎨 彩色笔记标签
- 💾 本地存储（无需联网）
- 🌐 当前界面语言：简体中文（后续版本将支持多语言）

## 下载
可点击[Releases]选择0.2.0版本下载源代码压缩包（Source code）

## 开发
- Electron + 原生 JS + nw.js
- 存储：electron-store
- 运行：`electron . （ electron . 要有 electron 全局安装在电脑，否则会报错）
