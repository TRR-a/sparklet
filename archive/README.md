\# sparklet 浏览器扩展

一个轻量的浏览器扩展工具，专注于笔记管理与界面交互优化，适配中文使用场景。



\## 📌 核心功能

\- 深浅色模式切换：一键切换界面风格，适配不同光线环境

\- 多笔记管理：支持创建、存储、编辑多条笔记，本地数据持久化

\- 回收站功能：误删笔记可一键还原，避免数据丢失；回收站支持清空/筛选

\- UI交互优化：新增关闭按钮、精简界面布局，仅支持中文显示，操作更贴合中文用户习惯



\## 📂 文件结构说明

sparklet/

├── images/          # 扩展所需图片资源（图标、背景、按钮素材等）

├── popup/           # 扩展弹窗页面（包含HTML布局、CSS样式、JS交互逻辑）

├── scripts/         # 扩展核心业务脚本（笔记存储、深浅色切换、回收站逻辑等）

├── generate\_icons.py # 扩展图标批量生成辅助脚本（可选）

└── manifest.json    # 浏览器扩展核心配置文件（必选，定义扩展名称/权限/版本等）



\## 🚀 安装方法（Chrome/Edge通用）

1\. 打开浏览器，在地址栏输入 chrome://extensions/ 并回车，进入扩展程序页面；

2\. 开启页面右上角的「开发者模式」（开关按钮）；

3\. 点击页面左上角的「加载已解压的扩展程序」；

4\. 选择本地的sparklet根文件夹，点击「确定」即可安装。
